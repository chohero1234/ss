## 시스템 구조

```
[카메라]
   │
   ▼
[PC]
  ├─ Vision AI 모델 (MobileNetV2)
  │    └─ 객체 인식 결과 출력
  │
  └─ 시리얼 통신 (USB)
         │
         ▼
    [Arduino]
      ├─ 서보모터
      └─ LED
```
> 위 다이어그램을 실제 시스템에 맞게 수정하세요. 필요하다면 이미지 파일로 대체할 수 있습니다.

## 사용 기술 및 부품

### 소프트웨어
| 항목 | 내용 |
|------|------|
| Vision AI | MobileNetV2 / Pytorch / OpenCV|
| 언어 | Python 3.x |
| 아두이노 IDE | v2.x |

### 하드웨어
| 부품 | 수량 | 용도 |
|------|------|------|
| Arduino Uno  | 1 | 메인 컨트롤러 |
| 웹캠 | 1 | 영상 입력 |
| 서보모터 SG90 | 2 | 물체 이동 제어 |
| LED (Red/Green) | 각 1 | 상태 표시 |
| 브레드보드, 점퍼선 | - | 회로 구성 |

## 구현 방법

### 1. Vision AI 모델

- **모델 선택 이유**: (실시간 추론 속도가 빠르고 경량화가 용이하여 선택)
- **데이터셋**: (직접 촬영 / 공개 데이터셋 사용 여부 및 출처)
- **학습 방법**: (Transfer Learning, Fine-tuning 등)


### 2. 아두이노 제어 로직

- 수신된 명령어에 따라 서보모터 각도 제어
- LED 신호로 시스템 상태 시각화
- (회로도 이미지가 있다면 삽입)
